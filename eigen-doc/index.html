这是一个Eigen3的API文件. 你可以 下载 它，以一个tgz文件的形式进行离线的阅读。

第一次接触 Eigen，最好看一下 入门指南 页面，此页面告诉你如何书写和编译你使用 Eigen 的第一个程序.

然后， the quick reference pages 以一种很紧凑的格式给你一个非常完整的API描述，这对于回想特定feature的语法或快速浏览API非常有用。它们目前涵盖了以下两个feature集，未来还会有更多:

  • [QuickRef] 稠密矩阵和数组的操作
  • [QuickRef] 稀疏线性代数

你是一个 MatLab 使用者么? 这里还有一份 short ASCII reference 可以用于 Matlab 的转译.

The main documentation 被组织到 章节 中，涵盖了不同领域的features. 他们本身以全面的方式描述不同features的 user manual 页面和允许你通过相关的 Eigen 的 modules 和 classes 访问API文档的 reference 页面组成。

在 扩展/定制 Eigen 章中，你将会找到关于扩展 Eigen features 和支持自定义标量类型的讨论和例子。

在 通用话题 章中，你将会找到关于更多通用话题（如预处理器指令、控制assertion、多线程、MKL支持、一些 Eigen 的internal insights等等）。

最后，不要错过搜索引擎，它对于快速获取一个给定 class 或 function 的文档很有用。
